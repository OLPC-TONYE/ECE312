Introduction

What we're doing

According to the glossary of Charlson and Hertzenberg (1985), Soot are: “Particles formed during the quenching of gases at the outer edge of flames of organic vapour consisting predominantly of carbon, with lesser amounts of oxygen and hydrogen present as carboxyl and phenolic groups and exhibiting an imperfect graphitic structure.” In shorter words, It is a mass of impure carbon particles resulting from incomplete combustion of hydrocarbons. Soot is regarded as one of the highest contribution to air pollution and by how much fuel(hydrocarbons) we burn each day, it is easy to see why. Research is done to see how we can reduce how much of it we release into the atmosphere. So any information on soot is welcomed.

Conclusion

What we're doing

Python was easily our choice for a language for its variety of frameworks and libraries. Among these libraries, Django was the first to come to mind since it is an API that aids web backend development. Coding had to be done across multiple devices so Github was also used for source control. For our project we chose to follow the prototype model, so there can always be a ‘prototype’, and be able to modify if new information comes in.
